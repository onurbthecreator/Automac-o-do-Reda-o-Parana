🎓 ONURB – Automação de Redação Paraná

Sistema web que automatiza o processo de envio de redações do Redação Paraná, utilizando JavaScript, HTML e CSS, com suporte opcional a Inteligência Artificial (IA) para gerar textos automaticamente.

O objetivo é facilitar e acelerar o envio de redações diretamente pelo navegador.

🚀 Demonstração

Interface moderna com:

Fundo animado

Sistema em etapas

Integração com IA

Console de debug


🧠 Funcionalidades

🔐 Autenticação com Bearer Token

📋 Listagem automática de redações disponíveis

🤖 Geração de redação com IA (Groq, OpenAI ou Claude)

📝 Editor de texto integrado

📤 Envio automático da redação

📊 Barra de progresso do envio

⚙️ Como Usar
1. Abrir o site

Basta abrir o arquivo:

index.html

ou hospedar no:

GitHub Pages

Vercel

Netlify

2. Pegar o Token do Redação Paraná

Acesse:

https://redacao.pr.gov.br

Faça login com sua conta Google

Pressione:

F12 → Network → Authorization

Copie o Bearer Token

Ele começa com algo parecido com:

eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
3. Colar o Token no Sistema

No site:

Clique em Iniciar

Cole o token

Clique em Conectar

🤖 Usar Inteligência Artificial (Opcional)

Você pode usar:

Groq (gratuito)

OpenAI

Claude

Basta adicionar sua API Key.

Se não adicionar, o sistema usa um texto exemplo.
